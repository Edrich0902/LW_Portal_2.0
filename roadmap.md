# LW Portal 2.0 - Future Roadmap

This document outlines the planned features and enhancements for Phase 2 and beyond. These features aim to transition the portal from a content management tool to an operational intelligence and engagement platform.

## 1. Content Scheduling & Automation
- **Scheduled Announcements**: Allow users to set a `publish_at` date for announcements to be sent automatically.
- **Push Notification Integration**: Integration with Firebase or similar services to alert users when announcements are published.

## 2. User Engagement & Lifecycle
- **Membership Workflow**: Track membership and baptism progress through automated status updates and notes.
- **Member Journey Timeline**: A visual timeline on user profiles showing key milestones (First Visit, Membership Class, Baptism).

## 3. Analytics & Reporting
- **Custom Export Filters**: Advanced filtering for CSV exports (e.g., "All members baptized in the last 6 months").
- **Engagement Metrics**: Track views and interactions for Sermons and Announcements.

## 4. System Health & Integrity
- **Activity Logging (Audit Trail)**: A system-wide log tracking all create, update, and delete actions for accountability.
- **Global Search (Command Palette)**: Implement `Cmd+K` global search functionality for quick access to all entities (Users, Sermons, Events, etc.).
- **Granular Permissions**: Expand the role system (Admin, Editor, Viewer, Social Media Manager) with specific access controls.

## 5. Media Management
- **Media Library Screen**: A dedicated screen to browse, search, and reuse images uploaded to Cloudinary.
- **Image Metadata**: Support for alt-text, captions, and tags on uploaded media.

## 6. UI/UX Improvements
- **Bulk Actions**: Support for deleting or updating multiple items in the DataTables.
- **Interactive Dashboard Charts**: Visualizing user growth and event attendance over time.
